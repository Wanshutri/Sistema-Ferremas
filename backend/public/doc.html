Documentación de la API de Ferremas

¡Bienvenido a la API del sistema de Ferremas! Esta API está desarrollada en JavaScript, utilizando el entorno de desarrollo de Node.js y el framework Express. Se ejecuta en el puerto 3001 y se conecta a una base de datos MySQL Community Server. Se centra exclusivamente en la gestión de la tabla de productos. Por lo tanto, la funcionalidad principal de esta API abarca el CRUD de productos (Crear, Leer, Actualizar y Eliminar). A continuación, proporcionamos un resumen de las rutas disponibles, su funcionamiento y una descripción general de su estructura.

Método GET: Obtener Datos

Obtener todos los productos

Realiza una consulta a la tabla de productos y devuelve todos los productos registrados en un formato JSON. Si no hay ninguno, devuelve un arreglo vacío.

Ruta: http://localhost:3001/api/productos

[{"idProducto":1,"nombreProducto":"Taladro Bauker","precioProducto":42000,"stock":7}, {"idProducto":2,"nombreProducto":"Destornillador Philips","precioProducto":8000,"stock":14}]

Obtener un producto

Realiza una consulta a la tabla de productos y devuelve el producto que concuerde con el ID entregado como parámetro en :id en un formato JSON. Si no existe, devuelve un código de error 404.

Ruta: http://localhost:3001/api/productos/:id

Ruta de ejemplo: http://localhost:3001/api/productos/2

{"idProducto":2,"nombreProducto":"Destornillador Philips","precioProducto":8000,"stock":14}

Método POST: Enviar Datos

Insertar un producto

Realiza una inserción a la tabla de productos con los datos entregados en el cuerpo del request, devuelve el código 201 si lo hace con éxito.

Ruta: http://localhost:3001/api/productos

Ejemplo de formato del body (el ID se genera automáticamente)

{ "nombreProducto" : "ejemplonombre", "precioProducto" : 1, "stock" : 5 }

Método PUT: Actualizar Datos

Actualizar un producto

Realiza una actualización del contenido de un producto indicado a través y en el parámetro de :id. Devolverá 404 si no encontró el producto y éxito si lo actualizó correctamente.

Ruta: htfrontend/src/assets/img/logo.png { "nombreProducto" : "ejemplonombre", "precioProducto" : 1, "stock" : 5 }


Método DELETE: Eliminar Datos

Eliminar un producto

Elimina el producto indicado en y a través del parámetro :id, enviará código 404 si no lo encuentra y éxito si lo elimina correctamente.

Ruta: http://localhost:3001/api/productos/:id
